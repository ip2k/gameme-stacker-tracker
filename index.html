To use this page, specify the server:port in the URL fragment (the part after the #). Here are some examples:

  • HyperionGaming.org | Payload | West Coast: #66.151.138.182:27015
  • (-TN-) Payload Rotation Server: #107.6.75.68:27015
  • D .U .S .T .B .O .W .L - BEGINNERS - FRAGMASTERS.CO.UK: #87.117.217.32:27017

Note that the target TF2 server MUST support gameME stats for this page to work! If nothing shows up, it's likely that the specified TF2 server has no players!
